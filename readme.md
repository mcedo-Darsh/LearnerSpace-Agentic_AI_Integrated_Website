# Learner Space - Agentic AI Integrated Website
Learner Space is an initiative by Undergraduate Academic Cell (UGAC) of IIT Bombay which aims to introduce and give basic understanding of different domains to the students.

## Agemtic AI Integrated Website
During this course, students would be introduced to the process of web development. As well as using APIs to make an Agentic AI integrated website.

### Week 1
Here the web development journey begins. This week I dive into HTML & CSS, the backbone of every website on the internet. Made a simple login page using html and css.

### Week 2
This week, I explore javascript. This is the language that makes webpages functional. Every webpage, website on the internet is essentially has a html, a css and a js file. Made a task manager using html, css and js.

### Week 3
Agentic Ai into the project. This week, I learned about node.js, API keys, and use my Gemini API key to make a webpage. This is a smart task planner. Give it any task and it would make a task plan for you.

### Week 4
Final Project - **Finance tracker Website**

This week's task was to make a Finance tracker website which has multiple webpages including a landing page, dashboard, reports and an Ai assistant to discuss about your finances. This website even had a dark mode option.

About the website -

Landing page - A nice looking page to welcome the users.

Dashboard - This page gives the summary for all your finances. It shows balance, income, expense andd monthly budget. This page also shows recent transactions and expense breakdown.

Add transaction - This page is dedicated for adding transactions.

History - This page shows all the transactions that have been added to the account. Here the user can view and delete transactions.

Reports - This page shows expense distribution and features an option to view Ai insights for you expenses.

Ai Assistant - This is a dedicated page for Ai chatbot. Here the user can ask queries and get answers from an Ai.

Settings - This is the settings page, it gives a few options - Change monthly budget, change currency, dark mode and an option to delete all the previously added transactions.
